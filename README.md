## 1. プロダクト概要

**資産・口座・医療や葬儀の希望・家族へのメッセージを、誰のサーバにも送らず端末内だけで整理し、PDFとして遺せる、完全オフライン・買い切りの終活ノート。**

- カテゴリ：Life（終活／エンディングノート）
- 中核価値：非送信・買い切り・PDF書き出し（＝サービス終了しても端末と紙に残る）
- 対象：50〜60代（自分の終活／親の手伝い世代）
- 競合との差別化：無料クラウド勢（わが家ノート等）の弱点＝サービス終了・課金不安・プライバシーを正面から突く

**絶対原則**：完全オフライン／非送信・データ収集ゼロ・広告ゼロ・サブスクなし・Apple純正のみ・法的助言はしない。

## 5. 残課題（【要確認】統合）

1. J-PlatPat 商標確定照会（「ユズリ」区分9／11C01）、en版ブランド名。

3. 価格はローンチ後に再検証（¥1,200／¥1,800への調整余地）。
4. en を「汎用英語」か「en-US特化」か。各ロケール免責の法域差レビュー体制。
5. カスタム項目の上限・並べ替え・既定項目の非表示可否。
6. プライバシーポリシー文面（GitHub Pages掲示、要レビュー）。
7. 免責・医療/法律文言の専門家レビュー。
8. 提出直前に最新App Storeガイドライン再確認（3.1.x／4.2／1.4.1）。
